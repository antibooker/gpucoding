# CUDA VectorAdd Demo
基于 CUDA C 实现 GPU 并行向量加法，对比 CPU 与 GPU 在不同数据规模下的计算效率，熟悉 CUDA 核函数、内存管理与错误检查。

## 功能
- CPU 串行向量加法
- GPU 并行向量加法（CUDA 核函数）
- 多规模数据测试（1024 / 10240 / 102400）
- 耗时对比与结果验证
- CUDA 运行时错误检查

## 运行结果示例
===== 数据规模：1024 个元素 =====
CPU 计算耗时：0.00 ms
GPU 计算耗时：0.97 ms
计算结果验证：前2个元素结果为 0, 3

===== 数据规模：10240 个元素 =====
CPU 计算耗时：0.03 ms
GPU 计算耗时：2.67 ms
计算结果验证：前2个元素结果为 0, 3

===== 数据规模：102400 个元素 =====
CPU 计算耗时：0.26 ms
GPU 计算耗时：12.98 ms

Exit status: 0

## 技术要点
- CUDA 核函数编写与调用
- 主机/设备内存分配与数据拷贝
- 并行线程块/网格配置
- 计时与性能对比
- 错误处理机制
